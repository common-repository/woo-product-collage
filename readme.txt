=== Product Collage ===
Tags: Product Collage, Woo Product Collage, Collage, Wordpress Collage
Requires at least: 4.3
Tested up to: 4.8

== Description ==

Product Collage is a plugin used to create products collage. When you add a new collage then system will generate a code and you can use that code in any other website.

To view the code, on the listing page you just need to click on "View Code" button under "Action" tab.

You can then copy and paste that code in to other websites. Website having Collage code will then show all the products which you assigned during creating the Collage.

= Salient Features Of Product Collage: =
- Multiple responsive layouts to choose
- Quick image selection windows
- Easy drop down menus
- Multiple colour schemes
- Easy sharing

Clicking on any of the product images will redirect the user to product details page of the WooCommerce website on which the plugin is installed.

= Usage Guide =
- Select the category and the products to be added to the collage
- Select the layout
- Choose the colour scheme
- Click “Create”
- A collage will be created and a script will be provided that can be placed on web pages

This plugin has been developed to provide great collages for your WooCommerce website. Any suggestions or feature requests are welcome! 

== Installation ==
- Download the Plugin (in a zip file) on your system
- In the Admin Panel click on Plugin and select ‘Add New’ from the sub menu
- In the next screen Upload the plugin by browsing to the location where you have downloaded the plugin
- Activate the plugin through the 'Plugins' menu in WordPress
- Add image collages on the page or post screen

Or 

- Upload the Product-Collage directory (directory included) to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress
- Add image collages on the page or post screen
